# InThon-team-2  
**학부 연구생 ↔ 연구실 매칭 플랫폼 (Kunnect-style)**

## 📌 프로젝트 개요  
학부생이 연구실을 찾고, 교수/연구실이 적합한 학생을 모집할 수 있도록 돕는 플랫폼입니다.  
학생, 연구실, 학교(및 연구지원센터)를 위한 통합 솔루션으로,  
연구 참여 기회의 접근성 및 연구실의 모집 효율을 크게 개선하는 것을 목표로 합니다.

---

## 📌 개발 동기  
- 학부생은 연구 참여에 관심이 있으나  
  – 연구실 정보를 찾기 어렵고 “어디에 지원해야 하나?”, “내가 적합한가?”하는 고민이 많습니다.  
- 교수/연구실은  
  – 지원자 홍보/모집이 번거롭고 적합한 학생을 찾느라 시간을 많이 소비합니다.  
- 이러한 문제를 해결하기 위해 연구생과 연구실을 효율적으로 연결하는 플랫폼을 구축하고자 합니다.

---

## 📌 주요 사용자(타깃)  
- 학생: 연구 경험을 쌓고자 하는 학부생  
- 교수/연구실 관리자: 연구생을 모집하고자 하는 연구실  
- 학교/학과 연구지원센터: 학생-연구실 매칭 프로그램을 운영하는 기관

---

## 📌 주요 기능  
- 연구실 탐색 및 필터링: 연구 분야, 기술 스택, 지도 교수명 등으로 검색  
- 학생 프로필 생성: 학과, 기술 스택, 프로젝트 내역, 관심 분야 등록  
- 권한 관리: 학생/연구실 관리자 역할 분리  

---

## 📌 기술 스택  
- Frontend: React (+TypeScript)  
- Backend: Django + Django REST Framework  
- Database: MySQL/PostgreSQL

---

## 📌 기대 효과  
- 학생: 연구 기회 접근성 증가, 진로 탐색 기회 확장  
- 교수/연구실: 모집 절차 간소화, 적합한 학생 확보  
- 학교/학과: 학부연구 프로그램 운영 효율성 증대, 데이터 기반 정책 수립 가능
